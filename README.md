\# Return Risk Analyzer (MVP)



Visualize \*\*drawdowns, volatility, and “pain”\*\* from historical returns.  

No prediction. No recommendation.



This tool is designed to help you \*\*understand how painful an asset has been to hold\*\*,  

based purely on past data.



---



\## What this tool does



\- Fetches historical prices (Yahoo Finance)

\- Calculates risk-focused metrics:

&nbsp; - Max Drawdown

&nbsp; - Drawdown Duration

&nbsp; - Daily Volatility

&nbsp; - Worst Days

\- Aggregates them into a single \*\*Pain Score (0–100)\*\*  

\- Visualizes:

&nbsp; - Equity Curve

&nbsp; - Drawdown Curve



> This is \*\*not\*\* a forecasting or signal-generating tool.  

> It is a retrospective risk visualization tool.



---



\## How to use



\### Option 1: Web UI

1\. Enter a ticker (e.g. `GLD`, `SPY`)

2\. Select a period (1Y / 3Y / 5Y / MAX)

3\. Click \*\*Analyze\*\*
